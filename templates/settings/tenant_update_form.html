Tenant Description

The Tenant model in the Django web application represents the company that is using the application. In this context, a Tenant refers to "Mala firma j.d.o.o." or any other company that may use the app.

This model includes essential information such as the company name, address, contact information, and relevant business details.

The Tenant model helps in isolating data and settings for each company, ensuring that each tenant's data is securely separated and managed independently.

This is crucial for maintaining data privacy, providing tailored features, and offering a customized user experience for each company using the application.
